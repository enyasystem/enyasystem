## 🔐 Security-Aware Development

I hold a completed ALX Cybersecurity certification and actively integrate security practices into every layer of my development work.

**Applied security work includes:**
- Webhook signature verification — Stripe and Paystack
- JWT authentication with token rotation, expiry, and refresh logic
- Rate limiting, IP blacklisting, and request throttling (Node.js, Django)
- DNS security hardening — SPF, DKIM, DMARC configuration
- Docker container isolation and secrets management
- CI/CD pipeline security controls — GitHub Actions, Jenkins
- Environment variable security and .env audit practices

**Currently expanding into:**
- Kubernetes security and container orchestration (Linux Foundation)
- Infrastructure as Code security practices
- DevSecOps pipeline integration
